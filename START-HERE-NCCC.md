# Start Here for NCCC Students

This repository contains the course materials.

For the hands-on lab, create your own private project from the Project VIC Agentic AI Development Project Template:

<https://github.com/Project-VIC-International/Agentic-AI-Development-Project-Template>

During the lab, your private project repository is where you will:

1. Complete the problem intake form.
2. Ask Cursor to review the project rules.
3. Create a milestone plan.
4. Generate specs, plans, and tasks.
5. Build and test with synthetic data.

Do not paste real case data into Cursor, GitHub, issues, logs, screenshots, test files, or sample data. For this course, all development must use synthetic data.
