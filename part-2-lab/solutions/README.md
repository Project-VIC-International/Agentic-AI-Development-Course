# Solutions

This directory contains reference solutions for the lab exercises.

Solutions are intentionally not provided for all exercises — the goal is for students to work through the exercises with the AI agent, not to copy pre-built answers.

## Available References

| Exercise | Reference | Notes |
|----------|-----------|-------|
| 04 - CASE/UCO Introduction | [../exercises/04-case-uco-introduction/examples/](../exercises/04-case-uco-introduction/examples/) | The example files serve as the reference |
| 05 - CAC Ontology Modeling | Generated during class | Each class produces a unique graph |
| 06 - Build a Prototype | Generated during class | Each class builds a different tool |

## Why Limited Solutions?

1. **The process matters more than the product.** The goal is to learn the workflow (spec → plan → build → validate), not to produce a specific artifact.

2. **Every class is different.** The prototype built in Exercise 6 depends on what the class chooses to build.

3. **AI output varies.** The agent may produce different but equally valid solutions each time.

4. **Copying defeats the purpose.** The course teaches investigators to guide an AI agent — that skill comes from practice, not from reading a solution file.

## For Facilitators

If you need reference solutions for grading or preparation:
1. Run through the exercises yourself before the class
2. Save the outputs as your reference
3. Keep in mind that student outputs will differ — evaluate the process and compliance, not exact match
