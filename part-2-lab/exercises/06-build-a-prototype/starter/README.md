# Prototype Starter

This directory is the starting point for the prototype you'll build in Exercise 6.

When the class chooses a tool to build, the agent will create files here during the build process.

## Expected Structure After Build

```
starter/
├── README.md              # This file (will be updated by the agent)
├── requirements.txt       # Python dependencies
├── src/                   # Source code
│   └── main.py           # Entry point
├── tests/                 # Automated tests
├── examples/              # Example input/output
│   ├── input/            # Sample input data (synthetic)
│   └── output/           # Expected output (CASE/UCO/CAC graphs)
└── docs/                  # Documentation
```
